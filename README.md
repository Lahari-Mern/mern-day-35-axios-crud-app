# Day 35 - Axios & CRUD Requests

REST API built with Node.js + Express. Tested with Postman.

## Tech Used
- Node.js
- Express.js
- Postman

## CRUD Operations
| Operation | Method | URL |
|-----------|--------|-----|
| Read | GET | /users |
| Create | POST | /users |
| Update | PUT | /users/:id |
| Delete | DELETE | /users/:id |

## How to Run
cd backend
npm install
node server.js

Server runs on http://localhost:5000
